# DBR進捗管理 マニュアル

## このアプリについて

DBR進捗管理（DBR Logger）は、[DBR難易度表](https://dbr-difficulty.github.io/index.html)との共同制作により開発された、beatmania IIDXの「DBR専用」プレー記録管理アプリです。

DBR難易度表に掲載されている曲データを基準に、クリアランプ・BP・スコアをまとめて管理できます。

**データの保存先について**
記録データはブラウザの`localStorage`に保存されます。端末やブラウザをまたいだ自動同期はありません。ブラウザのデータを消去すると記録も失われます。定期的にCSVで書き出しておくことを推奨します。

## はじめかた

1. **難易度表を読み込む**
   ページ上部の「難易度表読み込み」を押します。読み込みが完了すると、DBR難易度表のデータをもとに曲一覧が表示されます。前回の読み込みから12時間以上経過している場合、ボタンが目立つ色で表示されます。
2. **DBR難易度表の記録を取り込む（任意）**
   DBR難易度表対応形式のJSON記録をすでにお持ちの場合は、「JSON読み込み」で取り込むことができます。
   JSON読み込みの詳細は、「[データを読み込む・書き出す](#section-data)」をご参照ください。
3. **曲を探す**
   画面上部（スマホ）または画面右下（PC・タブレット）のQUICKFILTERで絞り込み軸を切り替えて、目的の曲を探します。
4. **曲を選択する**
   曲カードをクリック（タップ）すると曲が選択され、記録入力フォーム・推移グラフ・履歴が表示されます。
5. **クリアランプ・BP・スコア・メモを入力する**
   クリアランプ・BP・スコア・メモを入力します。すべてを入力する必要はありません。
6. **保存する**
   「保存する」を押します。クリアランプ・BP・スコアのいずれかを入力した場合はプレー履歴として記録されます。メモのみを入力して保存した場合は記録は作成されず、メモだけが保存されます。
7. **曲カード・推移グラフ・履歴で記録を確認する**
   保存した記録は曲カードのBP・スコア表示、推移グラフ、履歴に反映されます。
8. **バックアップ用に、必要に応じてCSVを書き出す**
   記録が増えてきたら、定期的に「CSV書き出し」で記録を保存しておくことを推奨します。端末やブラウザを変える際にも、CSVを使って記録を移行できます。

## 曲一覧を見る

曲一覧では、右上のボタンを押すことでカードビュー、リストビュー、テーブルビューを順番に切り替えられます。

### カードビュー

1曲ごとに情報をまとめたカードを表示します。プレーを記録すると、対応したカードの左側にクリアランプの色が表示されます。

| 表示項目 | 内容 |
| --- | --- |
| DBRLv. / SPLv. / 片手Lv. | 各譜面の難易度に関する情報です。 |
| クリアランプ | 保存済み記録の中で最上位のクリア状況です。 |
| BP | 最小BP / 最新BPを表示します。 |
| スコア | 最高スコアレート / 最新スコアレートをスクラッチ抜きの実質値で表示します。DBR難易度表にスクラッチ枚数が登録されていない譜面では※マークが表示されます。 |
| 最終プレー日 | 最後に記録した日付（月/日）です。 |
| プレー回数 | 保存されているプレー履歴の件数です。 |
| おすすめ度 / メモ | 曲名の下に表示されます。 |

DBR難易度表で新規提案中の譜面は、カードが黄色背景になり「新規提案中」バッジが表示されます。

### リストビュー

曲名・難易度・おすすめ度・メモと、現在のソートモードに応じた情報（BP、スコア、日付など）を一覧形式で表示します。

### テーブルビュー

多数の曲を比較しやすい表形式で表示します。表が画面幅を超える場合は、左右にスクロールできます。横スクロール中もクリアランプ色の帯、DBR、title列は左側に固定されます。

| 列 | 内容 |
| --- | --- |
| DBR / SP / 片手 | DBRLv.、SPLv.、片手Lv.です。 |
| ver / diff | 収録バージョンと譜面難易度です。 |
| title | 曲名です。メモがある場合は2行目に表示されます。 |
| BPM / rec / lamp | BPM、おすすめ度、クリアランプです。 |
| BP | 最小BP / 最新BPを表示します。 |
| score | 最高スコアレート / 最新スコアレートを表示します。 |
| last / plays | 最終プレー日と履歴件数です。 |

### テーブルビューの並び替え

- 列名ヘッダを選択すると、その列の昇順・降順を切り替えられます。
- テーブルビューでは、曲一覧上部の並び順プルダウンとページャの昇順・降順ボタンは表示されません。
- BP列は、最小BPの昇順 → 最小BPの降順 → 最新BPの昇順 → 最新BPの降順、の順に切り替わります。
- score列は、最高スコアの昇順 → 最高スコアの降順 → 最新スコアの昇順 → 最新スコアの降順、の順に切り替わります。
- 上部ページャの「？」ボタンを押すとランダム順に並び替えられます。

### ページ切り替え

カードビューとリストビューは1ページあたり100件、テーブルビューは1ページあたり500件表示します。表示件数を超える場合は前ページ・次ページで移動できます。

## 記録する

曲を選択すると、記録入力フォーム（Entry）・推移グラフ・履歴をまとめたメニューが画面下から開きます。別の曲を選択すると、メニューは先頭位置から表示されます。

### メニューを閉じる

- メニュー右上の「×」、またはEntry内の「閉じる」を押します。
- タッチ環境では、メニューが先頭位置にあるときに下方向へスワイプして閉じることもできます。

### 入力と保存

- クリアランプ・BP・スコア・メモをそれぞれ入力します。すべてを入力する必要はありません。
- クリアランプ・BP・スコアのいずれかを入力して保存すると、プレー履歴として記録されます。

### BP入力

- 通常はBPを直接入力します。
- 入力モード切り替えリンクを押すと「BAD/POOR入力」に切り替えられます。BADとPOORをそれぞれ入力すると、合計をBPとして保存します。
- BAD/POOR入力では、BADとPOORのどちらか一方だけを入力した状態では保存できません。両方入力するか、両方空にしてください。

### メモ

- メモは1曲につき1件、上書きで保存されます（最大200文字）。
- クリアランプ・BP・スコアを入力せずにメモだけ保存することもできます。この場合、プレー履歴は作成されません。
- メモを空にして保存すると、登録済みのメモが削除されます。

## 推移グラフ

曲を選択すると、その曲のBP推移グラフとスコア推移グラフが表示されます。

### BP推移グラフ

- BP値が入力された記録を時系列で表示します。BP未入力の記録はグラフに反映されません。
- 同じ日に複数の記録がある場合は、その日の最小BP値の記録が使用されます。

### スコア推移グラフ

- スコア値が入力された記録を時系列で表示します。スコア未入力の記録はグラフに反映されません。
- 同じ日に複数の記録がある場合は、その日の最高スコアの記録が使用されます。
- DBR難易度表にノーツ情報が登録されている譜面では、スクラッチ抜きの値を基準とした実質スコアランクの目安となる基準線がグラフ内に表示されます（AAA・AA・A）。

### グラフの操作

- グラフ上でマウスカーソルを動かすと、最も近い点の値と日付が表示されます。
- タッチ操作では、点をタップすることでその記録の情報が確認できます。

## 履歴を確認する・削除する

曲を選択すると、保存済みのプレー履歴一覧が表示されます。

各行の削除リンクを押すと、履歴を個別に削除できます。削除した記録は元に戻せません。

## 絞り込む（QUICKFILTER）

画面上に常に表示されるQUICKFILTERで、曲一覧の絞り込みができます。絞り込み軸ごとに、前回選んだ条件（ソート条件含む）が記憶されます。

### 絞り込み軸

| 軸 | 内容 |
| --- | --- |
| SPLv. | SP基準の難易度で絞り込みます。 |
| DBRLv. | DBR難易度表基準の難易度で絞り込みます。 |
| 片手Lv. | 片手難易度表基準の難易度で絞り込みます。片手難易度表に載っていない曲は一覧から除外されます。 |
| バージョン | 曲の収録バージョンで絞り込みます。 |
| BPM | BPM帯で絞り込みます。 |
| プレー日 | 記録した日付で絞り込みます。 |
| 曲名 | 曲名で検索します。表記ゆれや一部の記号違いも吸収して検索します。 |
| メモ | 保存済みメモの内容で検索します。 |

### 数値軸の単一選択・範囲選択

- SPLv.・DBRLv.・片手Lv.・バージョン・BPMでは、スライダーで値を選びます。
- 「範囲選択」を押すと、2点のスライダーで範囲を指定できます。
- 単一/範囲の状態は軸ごとに記憶されます。

### プレー日での絞り込み

- 単一選択では1つの日付の記録を、範囲選択では指定した期間の記録を表示します。
- 範囲選択の初期値は、記録がある日付のうち直近14日分です。
- 「前日」「翌日」ボタンを使うと、記録がある日付の前後に素早く移動できます。記録がない日はスキップされます。
- 「戻す」を押すと、直近の日付または範囲に戻せます。

### 曲名・メモ検索の「解除」

曲名またはメモ軸で絞り込んでいる時、「解除」ボタンを押すことで直前の軸に戻せます。

## 環境設定

全体サマリー内の「環境設定を開く」ボタンをクリックすると設定パネルが開きます。環境設定では、曲一覧に表示する対象や表示項目を切り替えられます。

### 表示項目

- **すべて**: クリアランプとBP・スコアの両方を確認できます。
- **クリア**: クリアランプとBPを中心に表示します。
- **スコア**: スコアランクとスコアを中心に表示します。

### 曲データ

AC、INFINITAS、AC限定（ACのみに収録）、INFINITAS限定（INFINITASのみに収録）、CS限定/削除曲のみ、で表示対象を絞り込めます。

プレー日で絞り込んでいる間は、曲データの設定は曲一覧に反映されません。

### 未査定曲

DBR難易度表の未査定曲を表示するかどうかを設定します。

プレー日または曲名・メモで絞り込んでいる間は、この設定は曲一覧に反映されません。

### 譜面難易度

B / N / H / A / L（BEGINNER / NORMAL / HYPER / ANOTHER / LEGGENDARIA）の譜面をそれぞれ表示するかどうかを設定します。

プレー日で絞り込んでいる間は、この設定は曲一覧に反映されません。

## 並び替え

曲一覧上部の並び順プルダウンで表示順を変更できます。「▲」「▼」ボタンで昇順・降順を切り替えられます。並び順は絞り込み軸ごとに記憶されます。

### 選べる並び順

曲名、バージョン、譜面難易度、SPLv.、DBRLv.、片手Lv.、BPM、おすすめ度、クリアランプ、最小BP、最新BP、最高スコア、最新スコア、最終プレー日、プレー回数、メモ、ランダム

表示項目が「クリア」のときはスコア系の並び順、「スコア」のときはBP系の並び順は選べません。

### 譜面難易度ソート

譜面難易度順で並び替えているとき、一覧の先頭に来る難易度を切り替えられます。ボタンを押すたびに B → N → H → A → L の順でローテーションします。

### おすすめ度ソート

おすすめ度順で並び替えているとき、一覧の先頭に来るおすすめ度を切り替えられます。ボタンを押すたびに ☆ → ◎ → ○ → △ → － の順でローテーションします。

## サマリーを見る

全体サマリーでは、現在の絞り込み条件に合わせて、難易度帯ごとの内訳と件数を帯グラフで確認できます。

### 帯グラフ

- SPLv.・DBRLv.・片手Lv.では、難易度帯ごとのクリアランプ（またはスコアランク）の内訳を表示します。
- プレー日での絞り込み中は、1本の帯が1日の記録を表します。
- 表示項目が「すべて」のとき、サマリー上部のプルダウンでクリアランプ集計とスコアランク集計を切り替えられます。
- バージョン軸で絞り込んでいるときのみ、譜面難易度による絞り込みボタンが表示されます。

### クリアランプ・スコアランクで絞り込む

- クリアランプやスコアランクのパネルをクリックすると、その区分の絞り込みをオン/オフできます。
- PCでは、パネルをダブルクリックすると、その区分だけを有効にした状態に絞り込めます。
- スマートフォンやタブレットでは、パネルを左にスワイプすると、その区分だけを有効にした状態に絞り込めます。

曲名またはメモで絞り込んでいる間は、クリアランプ・スコアランクによる絞り込みは無効になります。

## データを読み込む・書き出す

DBR Loggerのデータ入出力にはCSVとJSONの2種類があります。

- **CSV**: DBR Logger内のプレー履歴を保存・復元・端末間移行するためのものです。バックアップには定期的なCSV書き出しを推奨します。
- **JSON**: DBR難易度表対応形式の記録を取り込む・書き出すためのものです。DBR難易度表側のデータとのやり取りに使います。

### CSV書き出し

現在の全記録をCSVファイルとして書き出します。バックアップや端末移行の際に使用します。出力される列は次の通りです。

`timestamp` / `date` / `textageKey` / `title` / `level` / `splv` / `lamp` / `bp` / `score` / `memo`

- `level` はDBRLv.、`splv` はSPLv.です。
- プレー履歴はないがメモだけある曲も、メモ専用行として出力されます。

### CSV読み込み

DBR Loggerが書き出したCSVファイルを読み込みます。バックアップから記録を復元したり、端末間でデータを移行したりするときに使用します。

- 同一の記録が既に存在する場合は上書きされます。
- 読み込み後に取込件数と合計件数がアラートで表示されます。

### JSON書き出し

DBR難易度表対応形式のJSONファイルとして記録を書き出します。DBR IRへ反映したい場合は、後述する「DBR IR送信」機能を使うとよりスムーズです。

- BPは最小値、スコアは最大値、クリアランプは最上位ランプを出力します。

### JSON読み込み

DBR難易度表対応形式のJSONファイルを読み込みます。DBR難易度表側の記録をDBR Loggerに取り込む際に使用します。

- 読み込み時に記録日を指定できます。空欄の場合は今日の日付として読み込みます。

JSON読み込みでは、指定した記録日より前の既存記録と比較して、ランプ・BP・スコアのいずれかが改善している場合のみ履歴として取り込まれます。また、すでに反映済みの内容はスキップされます。

### DBR IR送信

「DBR IR送信」ボタンを押すと、DBR IRの受け取りページが開き、DBR Loggerに記録されたデータが自動的に送信されます。受け取りページでの操作はDBR IR側の案内に従ってください。

## 提案する

選択中の曲に対して、DBR難易度表への提案を送信できます。提案内容は外部スプレッドシートに送信されます。

| ボタン | 送信できる内容 |
| --- | --- |
| 新規提案 | レベル（必須）、おすすめ度、コメント、備考 |
| 変更提案 | 変更後レベル（必須）、提案理由（必須） |
| おすすめ提案 | 変更後おすすめ度（必須）、提案理由（必須） |

## テーマ切り替え

ページ上部のボタンで、ライトテーマとダークテーマを切り替えられます。選択したテーマはブラウザに記憶されます。

## キーボードショートカット

PCでは、キーボードからQUICKFILTERや曲の選択を操作できます。テキスト入力中はショートカットは動作しません。

| キー | 操作 |
| --- | --- |
| `Q` | QUICKFILTERを開閉します。 |
| `F` | DBRLv.軸に切り替えます。DBRLv.で絞り込み中にもう一度押すと、単一選択と範囲選択を切り替えます。 |
| `W` | SPLv.軸に切り替えます。もう一度押すと単一/範囲を切り替えます。 |
| `E` | 片手Lv.軸に切り替えます。もう一度押すと単一/範囲を切り替えます。 |
| `V` | バージョン軸に切り替えます。 |
| `B` | BPM軸に切り替えます。 |
| `R` | プレー日軸に切り替えます。プレー日で絞り込み中にもう一度押すと単一/範囲を切り替えます。 |
| `S` | 曲名検索軸に切り替えます。 |
| `T` | メモ検索軸に切り替えます。 |
| `A` / `D` | 日付を前後に移動、またはスライダーを前後に移動します。 |
| `Shift+A` / `Shift+D` | 範囲選択の終了日・終了値を移動します。 |
| 数字キー `1`〜`9` | 曲一覧のN番目の曲を選択します。 |
| `Delete` | 曲名・メモ軸の検索欄をクリアします。 |
| `Esc` | 検索欄のフォーカスを外します。選択曲のメニューが開いている場合は、メニューを閉じます。 |

## 全記録削除

ページ上部の「全記録削除」を押すと、保存済みのプレー記録とメモをすべて削除できます。難易度表データや表示設定は削除されません。

この操作は取り消せません。削除前にCSVで書き出しておくことを推奨します。

## 注意事項

- 記録データはブラウザの`localStorage`に保存されます。端末やブラウザをまたいだ自動同期はありません。ブラウザのデータ（Cookieやサイトデータ）を削除すると、記録も失われます。
- 記録の消失を防ぐために、定期的にCSV書き出しでバックアップを取ることを推奨します。端末やブラウザを変える前にも、CSV書き出しを行っておくことをお勧めします。
- 「DBR IR送信」を使用する際、ブラウザのポップアップブロックが有効な場合、受け取りページが開かないことがあります。その場合は、ブラウザのポップアップブロックの設定を確認してください。
- CSVやJSONの読み込み時に、曲名の表記が一致しない場合、正しく認識されないことがあります。

このアプリの使用により発生したデータ消失、記録の不整合、その他の損害について、作者は責任を負いません。重要な記録は、定期的にCSVで書き出して保管してください。

beatmania IIDX、INFINITAS、KONAMI、その他各権利者とは関係ありません。

### 各難易度表について

- **DBR難易度表**
  DBRで、ASSIST EASY/EASYゲージで80%以上残す、またはHARDゲージで完走のどちらか簡単な方を基準とした難易度表です。

- **片手難易度表**
  片手プレーで、HARDゲージでのクリアを基準とした難易度表です（オプション自由・あんみつ可）。

### 権利関係

- 曲名、譜面名、ゲーム内用語等の権利は、それぞれの権利者に帰属します。
- このアプリは、各権利者の権利を侵害する意図で公開しているものではありません。

### プライバシー

- このアプリは、入力されたプレー記録・BP・スコア・メモをブラウザの`localStorage`に保存します。
- 「DBR IR送信」を行うと、DBR IRの受け取りページにJSONデータを送信します。
- 提案フォームを送信すると、入力した提案内容が外部スプレッドシートに送信されます。
- 上記の機能を除き、このアプリは保存されたデータを外部に送信しません。
- GitHub Pages、ブラウザ、拡張機能、外部サービス等の挙動については、このアプリの管理範囲外です。
